# Sum of positive

## Description
You get an array of numbers, return the sum of all of the positives ones.
Note: if there is nothing to sum, the sum is default to 0

## Examples
```
    Input: [1, -4, 7, 12]
    Output:  20

    Explanation:  1 + 7 + 12 (without the -4)
```
